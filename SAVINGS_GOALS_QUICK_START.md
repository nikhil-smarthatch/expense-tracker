# Phase 1: Quick Start Guide - Savings Goals

## 🎯 What You Can Do Now

Create and track unlimited savings goals with real-time progress updates!

---

## 🚀 How to Use

### Step 1: Create Your First Goal

1. **Go to Income Screen** → Savings Goal section (see below)
2. **Tap "Create Goal"** button
3. **Fill in the form**:
   - **Goal Title**: "Emergency Fund", "Vacation", "House", etc.
   - **Description**: Why you need this (optional)
   - **Target Amount**: ₹1,00,000 (how much you want to save)
   - **Category**: Choose from list
   - **Priority**: High / Medium / Low
   - **Deadline** (Optional): If you want to reach goal by specific date
4. **Tap "Create Goal"**

### Step 2: Let the System Track It

The app automatically:

- ✅ Reads your monthly income (from Income entries)
- ✅ Reads your monthly expenses
- ✅ Calculates: **Available Savings = Income - Expenses**
- ✅ Updates your goal progress in real-time
- ✅ Shows if you're on track

### Step 3: Monitor Your Progress

**In Income Screen**:

- See your **Primary Goal** at the top
- Quick preview of progress
- Tap to see full details

**In Savings Goals Screen**:

- View all your goals
- See detailed progress for each
- Check deadline countdown
- Get status: ✅ On Track / ⚠️ At Risk / ❌ Behind

### Step 4: Edit or Complete Goals

- **Tap any goal** → Edit details or update amount
- **Delete goal** → Tap delete icon
- **Mark as complete** → Auto happens when target reached, or manual update

---

## 💡 How Status Works

The system automatically determines if you're on track:

```
✅ ON TRACK
   You're saving ≥ the required amount per month
   Example: Target ₹1,00,000 in 12 months = Need ₹8,333/month
            You're saving ₹10,000/month = ON TRACK ✓

⚠️ AT RISK
   You're saving between 80-99% of required
   Example: Need ₹8,333/month but saving ₹7,000/month
            Almost there, but slightly behind

❌ BEHIND
   You're saving < 80% of required
   Example: Need ₹8,333/month but saving ₹4,000/month
            Significantly behind

📌 NO DEADLINE
   No target deadline set - just track progress
   Shows amount saved but no "on track" status
```

## ⚡ Quick Tips

### Tip 1: Set Realistic Targets

- Consider your monthly income and expenses
- Set deadlines that make sense
- Don't stress if behind - adjust timeline anytime

### Tip 2: Use Categories Wisely

```
Emergency Fund → For emergencies (3-6 months expenses)
Vacation → Fun trip savings
House → Down payment / mortgage
Education → Courses or degree
Car → Vehicle purchase
Business → Startup capital
Retirement → Long-term savings
```

### Tip 3: Priority Better Organization

- **HIGH**: Must achieve (Emergency, Essential purchases)
- **MEDIUM**: Important but flexible (Vacation, Upgrades)
- **LOW**: Nice to have (Luxury items)

System shows HIGH priority goals first

### Tip 4: Track Real Transactions

- Every income entry helps progress
- Every expense affects available savings
- System is honest - it shows real progress!

### Tip 5: Adjust as Needed

- Deadline passed? Edit and extend it
- More income? Update savings plan
- Priorities changed? Reorder with priority setting

## ❓ FAQ

**Q: Can I create multiple goals?**  
A: Yes! Unlimited goals, each tracked independently.

**Q: What if I don't set a deadline?**  
A: That's fine! Still tracks current amount, just no monthly target.

**Q: What happens when goal is complete?**  
A: It moves to "Completed Goals" section with completion badge.

**Q: How does it calculate savings?**  
A: `Savings = Current Month Income - Current Month Expenses`
(Automatically pulled from your transactions)

**Q: Can I edit a goal?**  
A: Yes! Tap goal → Edit any field → Save changes

**Q: What if I fall behind?**  
A: Shows yellow warning. Extend deadline or increase income/reduce expenses.

**Q: Is my data safe?**  
A: Yes, stored securely on your phone with Hive database.

**Q: Can I delete a goal?**  
A: Yes, tap goal → Delete icon → Confirm
