# Expense Tracker App

An advanced Flutter-based mobile application for tracking personal expenses, income, and savings goals.

## Getting Started

This project helps users manage their finances with a clean, intuitive interface.

### Features

- **Expense Tracking**: Log daily expenses with categories, notes, and receipts.
- **Income Management**: Track income from various sources separately from expenses.
- **Savings Goals**: Create and monitor savings goals with automatic progress tracking based on income and expenses.
- **Image Attachments**: Attach and preview receipt images for any transaction.
- **Dashboard**: A comprehensive overview of your financial health.

### Prerequisites

- [Flutter SDK](https://docs.flutter.dev/get-started/install) (check `pubspec.yaml` for version)
- A configured IDE (VS Code, Android Studio)

### Installation

1.  **Clone the repository:**

    ```sh
    git clone <repository-url>
    cd expense-tracker
    ```

2.  **Install dependencies:**
    ```sh
    flutter pub get
    ```

### Running the Application

Run the following command to start the application on a connected device or simulator:

```sh
flutter run
```

For help getting started with Flutter development, view the
[online documentation](https://docs.flutter.dev/), which offers tutorials,
samples, guidance on mobile development, and a full API reference.
